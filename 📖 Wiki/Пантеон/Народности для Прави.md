# Народности для Прави

Это рабочий список народностей, которые берем как основу для мифологического атласа [[Фракции Прави|фракций Прави]].

Список нужен не для прямого переноса реальных народов в мир, а для выбора мифологических мотивов, образов богов, духов, героев, загробных представлений и небесных фракций.

## Примечания

- Этот список считается выбранным набором для дальнейшей работы над небесными фракциями.
- Для каждой народности не обязательно создавать отдельный пантеон. Некоторые будут объединяться в культурно-мифологические пласты.
- Если народность имеет несколько субэтнических или региональных ветвей, их можно использовать как отдельные источники мотивов.
- При переносе реальных мифов в мир Аал Лук Маас нужно сохранять уважительную дистанцию: брать мотивы, функции, образы и ритуальную логику, но создавать самостоятельные сущности сеттинга.
